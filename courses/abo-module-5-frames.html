Module 5: Ophthalmic Products: Frames

Learn frame materials, measurements, and selection for the NOCE exam.

Frame Materials

Zyl/Acetate: colorful, lightweight, hypoallergenic. Titanium: very light, strong, hypoallergenic. Stainless steel: durable, affordable. Memory metal: returns to shape. TR-90/Nylon: flexible, sports.

Key InsightTitanium is ideal for metal allergy patients and heavy prescriptions due to strength-to-weight ratio.

Frame Measurements

Box system: Eye size (A) x Bridge (DBL) x Temple. A = horizontal lens width. B = vertical height. ED = effective diameter (diagonal). DBL = distance between lenses.

Key InsightFrame PD = A + DBL. Should be close to patient PD for optimal optics.

Bridge Types

Keyhole: classic, sits higher. Saddle: distributes weight evenly across nose. Adjustable pads: precise fitting on metal frames. Low bridge fit: for flatter nose bridges.

Key InsightAdjustable nose pads allow precise vertex distance and tilt adjustments not possible with plastic bridges.

Temple Styles

Skull/library: curves behind ear. Comfort cable: wraps around ear (children/sports). Paddle: flat end. Spring hinge: allows flex, extends frame life.

Key InsightSpring hinges recommended for active patients and those with wider heads.

Frame Selection

Consider: Rx power, PD, face shape, lifestyle. High minus: smaller frames (edge thickness). High plus: larger eye size. Match frame PD to patient PD within 4mm for strong Rx.

Key InsightFrame selection impacts lens thickness, weight, and appearance significantly.
